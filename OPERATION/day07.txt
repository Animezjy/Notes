VPN(虚拟专用网)
在公用网络上建立专用私有网络

GRE VPN (只适用Linux)

一、启用GRE模块(client和proxy)
1)lsmod    //查看当前加载的模块

modprove ip_gre  //加载gre模块

二、配置
ip tunnel add tun0 mode gre remote 201.1.2.5 \
 local 201.1.2.10                                    //建立隧道

ip link set tun0 up   //启用隧道 

#为隧道tun0设置本地ip地址(10.10.10.10/24)
#隧道对面主机ip是(10.10.10.5)
ip addr add 10.10.10.10/24 peer 10.1010.5/24 dev tun0
#查看ip配置
ip addr show

PSSH (并发远程连接)
远程原理: expect  ssh 

pssh -i 将输出显示出来
-H host1 host2 host3  按主机连接
-h host.txt(h后面跟文件)
-p 指定并发连接数
-t 设置超时时间
-o dir 设置标准输出保存的目录
-e dir 设置错误输出信息保存的目录
-x     传递参数给ssh

pscp.pssh -r    批量拷贝

批量、多并发从其他主机下载数据到本机
pslurp 选项和pscp一致
